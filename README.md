# Breakout-Game
# 🎮 Breakout Game

A modern arcade-style Breakout game developed using HTML5 Canvas, JavaScript, and Object-Oriented Programming (OOP) principles.

---

# 🚀 Features

- Real-time gameplay using `requestAnimationFrame`
- Modular OOP architecture
- 3 progressive levels
- Strong bricks with multiple hit points
- Combo score multiplier system
- Power-up system
- Particle explosion effects
- Pause / Restart / Win / Game Over states
- Responsive canvas scaling

---

# 🕹️ Controls

| Key | Action |
|---|---|
| ⬅️ / ➡️ | Move Paddle |
| SPACE | Start Game / Next Level |
| ESC | Pause Game |
| R | Restart Game |

---

# 🎁 Power-Ups

| Type | Effect |
|---|---|
| 🟢 Paddle | Expands paddle width |
| 🔵 Slow | Slows down the ball |
| 🔴 Life | Grants extra life |

Power-ups spawn randomly when bricks are destroyed.

---

# 🧱 Game Mechanics

- The player must destroy all bricks to complete a level.
- The game contains 3 levels with increasing difficulty.
- Level 3 includes stronger bricks that require two hits.
- Players start with 3 lives.
- Losing the ball removes one life.
- The game ends when all lives are lost.
